CHAPTER 06

AI의 밥은 데이터!

데이터가 뭔지 알아보자 — AI가 배우려면 먼저 밥이 필요해요.

💡
핵심 정의

데이터(Data)란 관찰하거나 측정해서 얻은 사실과 수치의 모음입니다. AI에게 데이터는 말 그대로 밥이에요. 데이터 없이는 아무것도 배울 수 없고, 데이터가 많고 좋을수록 AI는 더 똑똑해집니다.

📝
정형 데이터 (Structured Data)

행과 열로 깔끔하게 정리된 표 형태의 데이터. 컴퓨터가 가장 다루기 쉬운 형태예요.

📊 예: 학생 성적표, 날씨 측정값, 주가 데이터
🖼️
비정형 데이터 (Unstructured Data)

정해진 형식 없이 자유롭게 존재하는 데이터. 현실에 존재하는 데이터의 약 80%가 이 형태예요.

📸 예: 사진, 동영상, 음성 파일, SNS 게시글
🕸️
반정형 데이터 (Semi-structured Data)

완전한 표 형태는 아니지만 일정한 규칙이 있는 데이터. 웹 서비스에서 자주 쓰여요.

💻 예: 이메일, JSON 파일, HTML 문서

데이터 품질이 왜 중요할까?

🗑️

쓰레기 입력 → 쓰레기 출력

나쁜 데이터로 학습한 AI는 엉터리 결과를 냅니다. "Garbage In, Garbage Out" — AI의 핵심 격언이에요.

⚖️

편향된 데이터의 위험

특정 집단 데이터가 부족하면 AI는 불공평한 판단을 해요. 채용 AI가 특정 성별을 불이익 준 실제 사례가 있어요.

📏

양보다 질이 먼저

100만 개의 나쁜 데이터보다 10만 개의 깨끗한 데이터가 낫습니다. 데이터 전처리가 AI 개발의 80%를 차지해요.

⭐
딱 한 줄 핵심!

AI의 성능은 알고리즘보다 데이터의 품질과 양에 더 크게 달려 있어요. 데이터를 잘 모으고 정리하는 것이 AI 개발의 출발점입니다.

CHAPTER 07

처음 만나는 머신러닝

AI가 스스로 배운다고? — 사람처럼 경험으로 익히는 원리를 알아봐요.

🤖
핵심 정의

머신러닝(Machine Learning)이란 컴퓨터가 명시적으로 프로그래밍되지 않아도 데이터에서 스스로 규칙을 찾아내어 학습하는 기술이에요. 사람이 "고양이는 귀가 뾰족하고…"라고 직접 알려주는 게 아니라, 수만 장의 고양이 사진을 보여줘서 AI 스스로 파악하게 합니다.

머신러닝의 3가지 학습 방식

🎯

지도 학습 (Supervised)

정답이 있는 데이터로 학습해요. 마치 선생님이 정답을 알려주며 가르치는 것처럼요.

예: 스팸 분류, 주가 예측
🔍

비지도 학습 (Unsupervised)

정답 없이 데이터의 패턴을 스스로 찾아요. 비슷한 것끼리 묶는 '군집화'가 대표적이에요.

예: 고객 그룹 분류, 추천 시스템
🎮

강화 학습 (Reinforcement)

보상과 벌칙으로 스스로 전략을 익혀요. 게임에서 이기면 +점수, 지면 -점수를 주며 학습해요.

예: 알파고, 게임 AI, 로봇 제어
💡
알아두기

현재 우리가 사용하는 유튜브 추천, 넷플릭스 추천, 챗GPT는 모두 지도 학습 또는 그 응용 방식을 기반으로 하고 있어요. 가장 널리 쓰이는 학습 방식이에요.

CHAPTER 08

딥러닝이 뭐길래?

사람의 뇌를 닮은 AI — 왜 딥러닝이 AI 혁명을 일으켰는지 알아봐요.

🧠
핵심 정의

딥러닝(Deep Learning)은 머신러닝의 한 종류로, 인간 뇌의 신경세포(뉴런) 연결 구조를 모방한 인공 신경망(Neural Network)을 여러 층(layer)으로 깊게 쌓아 학습하는 방식이에요. '딥(Deep)'은 신경망이 깊다, 즉 층이 많다는 의미예요.

인간 뇌 vs 인공 신경망

🧠 인간의 뇌🤖 인공 신경망
기본 단위뉴런 (신경세포)인공 뉴런 (노드)
연결시냅스가중치 (Weight)
학습경험으로 시냅스 강도 변화데이터로 가중치 값 조정
뉴런 수약 860억 개수백~수억 개 (모델에 따라)

신경망의 3층 구조

📥

입력층 (Input Layer)

데이터를 받아들이는 첫 번째 층. 이미지라면 각 픽셀의 밝기값이 입력돼요.

⚙️

은닉층 (Hidden Layer)

패턴을 찾고 변환하는 중간 층. 층이 많을수록 더 복잡한 특징을 학습할 수 있어요.

📤

출력층 (Output Layer)

최종 결과를 내보내는 층. "고양이 92%, 강아지 8%"처럼 확률로 답을 출력해요.

▲ 인공 신경망 구조 — 파란색: 입력층, 보라색: 은닉층, 초록색: 출력층
▲ 인공 뉴런의 작동 원리 — 입력×가중치의 합 → 활성화 함수 → 출력
🏆
역사적 사건 — 2012년 ImageNet 대회

토론토 대학 제프리 힌턴 팀의 AlexNet이 기존 최고 성능보다 오류율을 절반 이하로 낮추며 우승했어요. 이 사건이 딥러닝 혁명의 신호탄이 됐고, 전 세계 AI 연구 방향이 바뀌었습니다.

STEP 3. 다수결

① 훈련 데이터를 찍어주세요

예시: 빨강 3~4개를 왼쪽에, 파랑 3~4개를 오른쪽에 찍어보세요!
점 0개 찍힘
🟡 노란 점이란?
AI가 아직 정답을 모르는 새 데이터예요.
예를 들어 "고양이인지 강아지인지 모르는 새 사진"이라고 생각해 보세요.
이 노란 점을 캔버스에 찍으면, AI가 주변 이웃을 보고 A인지 B인지 판단해요.
👉 빨강·파랑 사이 경계 근처에 찍으면 결과가 더 재미있어요!
K값이란?
K = "몇 명에게 물어볼 것인가"예요.
노란 점 주변에서 가장 가까운 K개의 점을 골라, 빨강과 파랑 중 더 많은 쪽으로 분류해요.
K=1이면 가장 친한 친구 1명에게만 물어보는 것, K=5이면 5명에게 물어 다수결로 결정하는 거예요.
🔢 K값을 바꿔보세요

SUMMARY

📌 이 단원의 핵심 정리

01

데이터의 종류

정형(표), 비정형(이미지·음성·텍스트), 반정형(이메일·JSON). AI의 성능은 데이터 품질에 달려 있어요.

02

머신러닝 vs 일반 프로그래밍

일반 프로그램은 사람이 규칙을 만들고, 머신러닝은 데이터에서 AI가 스스로 규칙을 발견해요.

03

3가지 학습 방식

지도 학습(정답 O), 비지도 학습(정답 X), 강화 학습(보상 기반). 목적에 따라 다르게 사용해요.

04

딥러닝과 신경망

뇌의 뉴런을 모방한 인공 신경망을 여러 층으로 쌓은 것. 입력층 → 은닉층 → 출력층 구조예요.

05

KNN 분류 원리

"가장 가까운 K개 이웃의 다수결"로 분류해요. K값이 클수록 경계가 부드러워지고, 작을수록 예민해져요.
