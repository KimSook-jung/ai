🧠 핵심 개념

디지털 컴퓨터는 전기 신호의 켜짐(1) / 꺼짐(0)으로 모든 정보를 표현합니다. 이 한 자리를 비트(bit)라 하며, 8비트가 모이면 1바이트(byte)가 됩니다.

1 bit = 0 또는 1 1 byte = 8 bits 1 KB = 1,024 bytes 1 MB = 1,024 KB

💡 비트(Bit)와 이진수

컴퓨터는 모든 정보를 0과 1로만 저장합니다. 8비트 이진수에서 각 자리는 2의 거듭제곱 값을 가집니다.

📌 2^7=128, 2^6=64, 2^5=32, 2^4=16, 2^3=8, 2^2=4, 2^1=2, 2^0=1

🖱️ 직접 켜고 끄세요

비트를 클릭해 0 / 1을 바꿔보세요. 켜진 비트의 자릿값을 모두 더하면 십진수가 됩니다.

📖 이진수 덧셈 원리

이진수는 2가 되면 올림이 발생합니다. 1 + 1 = 10(이진수) → 십진수로 2입니다.

5 + 3 = 8
101 + 011 = 1000

🔄 십진수 → 이진수 변환 원리

십진수를 2로 나눠가며 나머지를 기록하고, 나머지를 역순으로 읽으면 이진수가 됩니다.

숫자를 입력하세요 (0~255)

나머지를 역순으로 읽은 결과

🔤 ASCII 코드란?

컴퓨터는 문자도 숫자로 저장합니다. 어떤 문자가 어떤 숫자에 해당하는지 정해놓은 표준이 ASCII입니다. 예: 'A' = 65, 'a' = 97, '0' = 48

📌 대문자 A~Z: 65~90 / 소문자 a~z: 97~122 / 숫자 0~9: 48~57

글자를 클릭해 이진수를 확인하세요

✍️ 문자 → 이진수 스트림

영문자를 입력하면 각 글자가 어떤 이진수로 저장되는지 보여줍니다.

🎨 RGB 색상 표현

모든 색은 빨강(R), 초록(G), 파랑(B) 3가지 빛의 조합입니다. 각 색상값은 0~255 범위로 8비트에 저장되며, 한 픽셀은 24비트(3바이트)를 차지합니다.

🖼️ 이미지와 픽셀

디지털 이미지는 수많은 픽셀의 집합입니다. 예를 들어 1920×1080 해상도 이미지는 1920 × 1080 × 3 = 6,220,800 bytes ≈ 5.9 MB의 공간이 필요합니다 (압축 전 기준).
